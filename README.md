# John Cena Secret Portal 🎺

A fun, hidden website with a secret unlock mechanism. When you first visit, you see a generic error page, but there's a way in...

## 🔐 The Secret

Enter the Konami Code to unlock the John Cena section:

```
↑ ↑ ↓ ↓ ← → ← → B A
```

(Arrow Up, Arrow Up, Arrow Down, Arrow Down, Arrow Left, Arrow Right, Arrow Left, Arrow Right, B, A)

## 🎮 Features

- **Error Page**: A generic error landing page with a hint
- **Secret Unlock**: Konami code activation
- **John Cena Section**: Displays when unlocked with games and proxy links
- **Vercel Ready**: Easily deployable to Vercel with one click

## 🚀 Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/john-cena-site.git
   cd john-cena-site
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.

## 📦 Building for Production

```bash
npm run build
npm start
```

## 🌐 Deploying to Vercel

### Option 1: Vercel CLI

1. Install Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

### Option 2: GitHub Integration

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import from GitHub
5. Select this repository
6. Click Deploy

Vercel will automatically detect it's a Next.js project and configure everything!
